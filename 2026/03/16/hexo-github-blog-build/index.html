Hexo + GitHub 博客搭建全记录｜从零开始搭建个人网站

发表于 2026-03-16 分类于 技术学习 阅读次数：

一、前言

这是我第一次使用 Hexo + GitHub Pages 搭建属于自己的个人博客，用来记录大学学习生活、技术笔记、成长感悟。

整个过程从零基础开始，一步步完成环境安装、主题配置、功能优化、部署上线，非常适合新手参考。

二、搭建流程

  1. 安装 Node.js 和 Git
  2. 安装 Hexo 脚手架
  3. 初始化博客项目
  4. 安装美化主题与插件
  5. 配置 GitHub 仓库
  6. 一键部署上线

三、核心命令

# 安装 Hexo
npm install -g hexo-cli

# 初始化博客
hexo init myblog
cd myblog
npm install

# 生成静态文件
hexo clean
hexo generate

# 本地预览
hexo server

# 部署到 GitHub
hexo deploy
